# 激光线位置对准软件

基于 **OpenCV + PyQt6** 的激光对准工具，从图像中检测标定板十字线（X/Y 轴及交点）和激光线，并实时显示：

- 激光线与 **Y 轴（竖直线）** 的夹角
- 激光中点到十字 **交点** 的距离（像素，可选 mm 标定）

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 使用说明

1. **打开摄像头** 或 **加载图片**
2. 将标定板十字线置于视野内，确保 X/Y 轴清晰可见
3. 打开激光，在 **激光检测参数** 中选择颜色（红/绿）并调节灵敏度
4. 界面左侧显示标注结果，右侧显示数值：
   - 橙色线 — X 轴（水平）
   - 青色线 — Y 轴（竖直）
   - 黄色十字 — 交点
   - 红色 — 激光线及中点
5. 如需物理距离，在 **像素/mm** 中填入标定值（0 表示仅显示像素）

## 检测原理

- **十字线**：Canny 边缘 + 霍夫线段检测，按角度聚类为水平/竖直两组
- **激光线**：HSV 颜色空间提取激光区域，`cv2.fitLine` 拟合直线
- **夹角**：激光线与 Y 轴方向的角度差（0~90°）
- **距离**：激光线段中点与十字交点的欧氏距离

## 注意事项

- 环境光过强时，请降低曝光或提高激光功率
- 十字线对比度不足时，检测可能失败，可增加标定板对比度
- Windows 摄像头使用 `CAP_DSHOW` 后端
